# Projeto Cadastro de Produtos

Este projeto foi criado com a ideia de cadastrar produtos e permitir que sejam listados para que você possa controlar o sistema. Ele é desenvolvido em Laravel PHP e utiliza o Tailwind CSS para estilizar as páginas, com o Blade como minha principal linguagem de modelagem HTML. Aqui, você pode cadastrar seus produtos e acompanhar as informações adicionadas a eles em uma tabela, com a opção de editar ou excluir o produto.
Utilizei o Wampp como servidor para o meu projeto e o MySQL que o mesmo me proporciona

# Iniciar o projeto

. composer install
O componser para instalar as dependencia do php laravel.

. npm install
o node está no projeto por conta do Tailwind css que estou usando para estilizar as paginas.

. Wampp
Usuado como servidor e bando de dados

# Banco de dados

O meu banco de dados foi iniciado dessa forma localmente

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=produtos
DB_USERNAME=root
DB_PASSWORD=

Estou usando uma migrate e para iniciar apenas ela precisa roda esse comando:

php artisan migrate --path=database/migrations/2023_10_18_184048_create_produtos_table.php

assim já vai ter acesso a tabela e os campos da tabela para que pode fazer o cadastro e pode ver todas as informação que está disponível.
e assim pode roda local usando php artisan serve [http://127.0.0.1:8000].
